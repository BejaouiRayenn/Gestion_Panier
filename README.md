# Application de Gestion de Panier d'Achat

Projet développé en PHP + PostgreSQL pour la gestion d'un panier d'achat complet.

## Fonctionnalités

- Consultation et recherche de produits
- Gestion du panier avec sessions PHP
- Vérification du stock en temps réel
- Formulaire client (nom, email, téléphone, adresse)
- Validation de commande avec transaction
- Ticket de commande imprimable
- Historique des commandes

## Technologies utilisées

- **Backend** : PHP 8
- **Base de données** : PostgreSQL
- **Connexion** : PDO
- **Frontend** : HTML5 + CSS3

## Installation

1. Cloner le projet
2. Créer la base de données `gestion_panier`
3. Exécuter le script SQL
4. Configurer `config.php` avec tes identifiants PostgreSQL
5. Lancer avec XAMPP/WAMP
